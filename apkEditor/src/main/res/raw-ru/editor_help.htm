Главное меню:

  • 1) Кнопка «Сохранить»: для сохранения изменений.
  • 2) Кнопка «Копировать»: для копирования выделенной строки в буфер обмена.
  • 3) Кнопка «Вставить»: для вставки содержимого из буфера обмена.
  • 4) Кнопка «Поиск»: для отображения или скрытия панели поиска.
  • 5) Кнопка «Ещё»: для отображения дополнительных параметров меню.
  • 6) HTML: для отображения содержимого в виде HTML-файла (только для чтения).
  • 7) Цветовая панель: для отображения диалогового окна цветовой панели, чтобы помочь вам получить правильный цвет.
  • 8) Фрагмент кода: для отображения диалога фрагмента кода. Доступно только при открытом smali-файле.
  • 9) Удалить строки: для удаления некоторых строк. При нажатии на кнопку появится диалоговое окно, в котором можно ввести значение строк «от» и «до», для удаления этих строк.
  • 10) Строки комментариев: комментировать или раскомментировать некоторые строки. Доступно только при открытом smali-файле.
  • 11) Настройки: отображение параметров настройки редактора.
  • 12) Помощь: открывает этот справочный документ.

    • Панель поиска:

  • 1) Входные данные для поиска строки
  • 2) Входные данные для замены строки
  • 3) Кнопка переключения регулярных выражений
  • 4) Кнопка переключения чувствительности к регистру
  • 5) Поле ввода номера строки
  • 6) Кнопка перехода. Если указан номер строки, переход к этой строке. В противном случае вставляется символ перевода строки \n, эквивалентный нажатию кнопки «Enter».
  • 7) Кнопка поиска.
  • 8) Кнопка замены. Просто замена текущего совпадения.
  • 9) Кнопка замены всего. Замена всех совпадений в текущем документе.
